Key Attributes of Backend MVC

  • Only responds to one event: an HTTP request
  • Stateless
  • Any action starts at the controller

Frontend MVC

Source: blah blah

Key Attributes of Frontend MVC

  • Responds to many different DOM events
  • Each layer has it's own state
  • Actions start in the view

Key Differences in Practice

Backend MVC Frontend MVC
Stateless & Static Contextually Interactive
Actions State + Actions
Slimmer controllers Heavier controllers
Direct function calls Event binding and callbacks
“Turn-based Strategy” “Real-time Strategy”
